Doom Facer v0.1

What is this page?

This is a help page for the doomfac.es API - if you're here it's because you navigated to help.html, or you caused an error with whatever parameters you used with doomfac.es. It's pretty hard to break if you're not being an idiot.

Usage:

http://doomfac.es/?health=100&scale=250&special=kill

You can use 'height' and/or 'width' in place of scale if you wish.

Health does sensible things between 0 and 100

Specials include: left, right, god, ouch, evil and kill

All the original faces (extracted from doom1.wad) are available here.
